Tugas 7: Form Login dengan AJAX

Nama: Ageng Prayogo

NRP: 5025241225

Kelas: A

Tugas ini bertujuan untuk membuat halaman login yang memvalidasi data pengguna secara asynchronous menggunakan AJAX.

Deskripsi & Konsep Implementasi

Aplikasi ini merupakan sebuah halaman login sederhana yang proses autentikasinya ditangani oleh AJAX (Asynchronous JavaScript and XML). Dengan AJAX, data dari form login dapat dikirim ke server dan responsnya dapat diterima kembali tanpa perlu me-refresh seluruh halaman. Ini menciptakan pengalaman pengguna yang lebih cepat dan modern.

Alur Kerja Aplikasi:

  1. Pengguna mengisi email dan password pada file login.html.
  2. Ketika tombol "Masuk" diklik, file login_ajax.js akan mengambil alih proses.
  3. JavaScript mengirimkan data email dan password ke server (check_login.php) di latar belakang menggunakan Fetch API.
  4. check_login.php menerima data, melakukan validasi sederhana (tanpa database), dan mengirimkan kembali status (success/error) dalam format JSON.
  5. JavaScript menerima respons JSON dan menampilkan pesan "Login Berhasil" atau "Email atau Password Salah" secara dinamis di halaman.

Tampilan utama form login yang akan divalidasi dengan AJAX.

Hasil Tugas

Halaman demo di bawah ini adalah hasil akhir dari pengerjaan tugas ini. Silakan coba login dengan data yang benar atau salah untuk melihat respons AJAX secara langsung.
