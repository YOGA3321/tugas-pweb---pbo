Tugas Profil dengan Semua Elemen

Nama: Ageng Prayogo

NRP: 5025241225

Kelas: A

Tugas ini bertujuan untuk membuat halaman profil pribadi menggunakan file HTML tunggal.

Deskripsi & Persyaratan

Sesuai instruksi, halaman profil ini harus dibuat dalam file `profil.html` dan wajib memuat elemen-elemen berikut:

  • Judul profil (menggunakan tag heading)
  • Paragraf singkat untuk deskripsi diri
  • Foto profil (menggunakan tag image)
  • Daftar hobi (menggunakan tag list)
  • Tabel riwayat pendidikan
  • Form kontak sederhana
  • Link yang mengarah ke media sosial

Hasil Tugas

Halaman demo di bawah ini adalah hasil akhir dari pengerjaan tugas ini, yang telah mencakup semua elemen yang disyaratkan.
