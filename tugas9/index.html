Tugas 9: Laporan Project (Aplikasi Restoran)

Nama: Ageng Prayogo

NRP: 5025241225

Kelas: A

Proyek ini bertujuan untuk membuat sistem autentikasi (Login & Register) yang aman dan modern untuk aplikasi restoran "Sederhana" menggunakan PHP, MySQL, dan AJAX.

Deskripsi & Konsep Implementasi

Aplikasi ini adalah fondasi dari sistem pemesanan makanan, dimulai dengan sistem autentikasi pengguna yang aman. Tampilan (UI) dibuat modern dengan *sliding panel* untuk beralih antara form login dan registrasi.

Sistem ini dibangun dengan memisahkan logika dari tampilan dan menerapkan beberapa konsep keamanan modern:

  1. Pemisahan Logika: Tampilan (`login.php`) dipisah dari logika pemrosesan (file PHP khusus di folder `auth/`).
  2. AJAX (Fetch API): Semua pengiriman form (login & register) ditangani oleh JavaScript (`auth.js`) secara *asynchronous* untuk pengalaman pengguna yang mulus tanpa *refresh* halaman.
  3. Notifikasi Interaktif: Menggunakan Pustaka `SweetAlert2` untuk memberikan umpan balik (sukses/error) yang elegan kepada pengguna.
  4. Password Hashing: Menggunakan `password_hash()` dan `password_verify()` untuk memastikan kata sandi disimpan dengan aman di database.
  5. Prepared Statements: Mencegah SQL Injection dengan menggunakan `prepare()` dan `bind_param()` pada semua query database.
  6. Google OAuth: Menyediakan opsi login alternatif yang aman menggunakan akun Google.

Alur Kerja Aplikasi (Login AJAX):

  1. Pengguna mengisi *username* dan *password* pada `login.php`.
  2. JavaScript (`auth.js`) mengambil alih proses *submit* form, mencegah *refresh* halaman.
  3. Tombol login menampilkan ikon "spinner" dan dinonaktifkan untuk mencegah *double-click*.
  4. Data formulir dikirim menggunakan Fetch API (AJAX) ke `auth/proses_login.php`.
  5. Server PHP memvalidasi data, melakukan *prepared statement* ke database untuk mencari *username*.
  6. Jika *username* ada, `password_verify()` digunakan untuk mencocokkan *hash* password.
  7. Server mengembalikan respons dalam format JSON (berisi status sukses/gagal, pesan, dan URL redirect jika sukses).
  8. JavaScript menerima JSON dan menampilkan notifikasi `SweetAlert2`. Jika sukses, halaman akan otomatis diarahkan ke dashboard.

Tampilan halaman login utama dengan form registrasi terpisah.
