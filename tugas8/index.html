Tugas 8: Pendaftaran Mahasiswa (PHP, MySQL, AJAX)

Nama: Ageng Prayogo

NRP: 5025241225

Kelas: A

Tugas ini bertujuan untuk membuat aplikasi CRUD (Create, Read) sederhana untuk pendaftaran mahasiswa baru menggunakan PHP, MySQL, dan AJAX.

Deskripsi & Konsep Implementasi

Aplikasi ini adalah sistem pendaftaran mahasiswa baru dalam satu halaman. Formulir pendaftaran dan tabel data pendaftar ditampilkan berdampingan. Pengguna dapat mendaftarkan mahasiswa baru, dan data akan disimpan ke database MySQL menggunakan PHP.

Halaman ini memanfaatkan AJAX untuk dua fungsi utama:

  1. Mengirim data formulir ke server PHP secara *asynchronous* (tanpa refresh) saat mendaftar.
  2. Mengambil data pendaftar terbaru dari server secara berkala (`setInterval`) untuk ditampilkan di tabel, memberikan ilusi pembaruan *real-time*.

Alur Kerja Aplikasi:

  1. Saat halaman dimuat, JavaScript (AJAX) memanggil api/get_jurusan.php untuk mengisi dropdown program studi.
  2. JavaScript (AJAX) juga langsung memanggil api/get_mahasiswa.php untuk menampilkan daftar mahasiswa yang sudah ada di tabel.
  3. Pengguna mengisi formulir pendaftaran (Nama, NIM, Email, Jurusan).
  4. Saat tombol "Daftarkan" diklik, JavaScript (AJAX) mengambil alih, mencegah refresh, dan mengirim data formulir ke api/add_mahasiswa.php.
  5. api/add_mahasiswa.php memvalidasi data, melakukan prepared statement untuk mengecek duplikat NIM, dan menyimpan data baru ke tabel mahasiswa.
  6. Server PHP mengembalikan respons dalam format JSON (berisi status sukses atau error).
  7. JavaScript menampilkan pesan sukses/gagal di bawah tombol. Jika sukses, form akan di-reset.
  8. Secara terpisah, fungsi setInterval di JavaScript akan otomatis memanggil api/get_mahasiswa.php setiap 5 detik, yang akan mengambil data terbaru (termasuk data yang baru saja ditambahkan) dan memperbarui tabel pendaftar.

Tampilan aplikasi pendaftaran dengan form di kiri dan tabel data di kanan.
